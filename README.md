## How to use

Clone this repository and cd into the project's folder:

```elm
git clone https://github.com/larafe1/pokedex.git && cd pokedex
```

Installing the project's dependencies:

```elm
-- Using npm:
npm install

-- Using yarn:
yarn
```

Running the app:

```elm
-- Using npm:
npm start

-- Using yarn:
yarn dev
```
